Metamodel files
---------------
 * Grafcet.ecore
 * Grafcet.km3
 * PetriNet.ecore
 * PetriNet.km3
 * PNML_simplified.ecore
 * PNML_simplified.km3
 * XML.ecore
 * XML.km3

Models' examples files
-----------------------
 * example.gf: grafcet example in TCS format
 * example.ecore: ecore version of the grafcet example (injected file)
 * Grafcet2PetriNet_example.ecore: result of the Grafcet2PetriNet transformation
 * PetriNet2Grafcet_example.ecore: result of the PetriNet2Grafcet transformation 
 * PetriNet2PNML_example.ecore: result of the PetriNet2PNML transformation 
 * PNML2PetriNet_example.ecore: result of the PNML2PetriNet transformation 
 * PNML2XML_example.ecore:  result of the PNML2XML transformation
 * XML2PNML_example.ecore:  result of the XML2PNML transformation
 * XML2Text_example.xml: result of the XML2Text transformation, corresponding XML text

Transformation files
--------------------
 * Grafcet2PetriNet.atl
 * PetriNet2Grafcet.atl
 * PetriNet2PNML.atl
 * PNML2PetriNet.atl
 * PNML2XML.atl
 * XML2PNML.atl
 * XML2Text.atl
 
"Grafcet.ecore", "PetriNet.ecore", "PNML_simplified.ecore" and "XML.ecore" are the metamodels for Grafcet, PetriNet, PNML and XML models.
Km3 files provide readable versions of these metamodels in the km3 format.
